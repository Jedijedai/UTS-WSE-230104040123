Project ini dibuat oleh Husni Majedi dengan NIM 230104040123 sebagai bagian dari Praktikum 7 mata kuliah Web Service Engineering. Aplikasi ini merupakan RESTful API sederhana menggunakan Express.js dengan resource utama bernama “orders”, sesuai ketentuan nomor akhir NIM. API ini menyediakan fitur CRUD untuk mengelola data pesanan, mulai dari menampilkan seluruh orders, mengambil order berdasarkan ID, menambah order baru, mengubah order, hingga menghapus order. Data disimpan dalam file JavaScript sebagai dummy data tanpa menggunakan database.

Pada project ini juga diterapkan beberapa aspek hardening API seperti penggunaan Helmet untuk meningkatkan keamanan header, CORS untuk mengatur asal request yang diperbolehkan, dan rate limiting untuk membatasi jumlah permintaan dalam jangka waktu tertentu. Monitoring dasar juga disediakan melalui endpoint health dan metrics yang menampilkan uptime, timestamp, jumlah request, serta penggunaan memori. Sistem logging diterapkan menggunakan Morgan yang mencatat semua aktivitas ke dalam console dan file access.log. Selain itu, tersedia pula endpoint khusus untuk menguji error internal server (status code 500) serta handler untuk menampilkan respons apabila route tidak ditemukan atau terjadi error di aplikasi.

Project ini dijalankan menggunakan Node.js, dimulai dengan menginstal semua dependensi menggunakan npm install lalu menjalankan server dengan perintah npm run dev. Seluruh endpoint dapat diuji melalui Postman maupun browser menggunakan URL dasar http://localhost:3000
. Project ini disusun untuk memenuhi kebutuhan pembelajaran terkait pembuatan layanan web yang aman, terstruktur, dan sesuai prinsip REST.
